SFFD Calls for Service Categories, Mar 2020

Encoding

The following categories for SFFD Service Calls are encoded as a Partition Layout - All Incidents, Call Type Group, Call Type and Unit Type. The hierarchy level of each node is encoded with color and vertical position. The width of each rectangle represents the fraction of records it comprises.

Interactivity

Hovering over the rectangles will display information about the incident grouping and highlight the ancestors.

Conclusion

From the visualization above, I can get a good sense of the weight of each category because the number of records is encoded into the rectangle width.

It can be seen that a near-majority of fire related cases are potentially life-threatening. Also, almost all potentially life-threatening cases are medical incidents.

A very small number of incidents are actually labeled under "fire".
